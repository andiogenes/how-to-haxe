Простой движок для браузера
Haxe и JavaScript | 25.05.2018 |

Одно из целевых применений Haxe - разработка игр. Так как язык похож на ActionScript и изначально опирался на работу с Flash (ведущий разработчик языка и лидер проекта Nicolas Cannasse изначально занимался флеш-играми и поддержкой технологии в компании Motion Twin), определенную долю его аудитории заняли Flash-разработчики, так как Haxe позволяет собирать под множество целевых платформ при помощи трансляции в код платформы, стало возможным использовать интерфейсы библиотеки AS3 для разработки под десктоп, мобильные устройства, HTML5 и т.д. Большой успех получил фреймворк OpenFL, представляющий перенос API Flash на Haxe, на его основе написаны порты игровых библиотек с AS3, такие как HaxePunk - порт движка FlashPunk, HaxeFlixel - порт Flixel. В последнее время большой интерес представляет фреймворк Kha, созданный с целью создания единой базы для разработки под всевозможные платформы, с интерфейсами для работы как на высоком уровне, так и на низком. Глубже со всеми этими библиотеками мы познакомимся чуть позже.

Ray casting, рейкастинг, Метод «бросания лучей» (англ. ray casting — рус. бросание лучей) — один из методов рендеринга в компьютерной графике, при котором сцена строится на основе замеров пересечения лучей с визуализируемой поверхностью. Этот термин впервые использовался в компьютерной графике в 1982 году в публикации Скотта Рота (англ. Scott Roth), который применил его для описания метода рендеринга CSG-моделей. - Википедия

Введение в Haxe
Введение в Haxe | 17.04.2018 |

В широком смысле Haxe (Haxe Toolkit) – это кросс-платформенный инструментарий с открытым исходным кодом, который содержит в себе следующие вещи:

  • Сам язык. Haxe – это высокоуровневый язык программирования с строгой типизацией, который транслируется «кросс- компилятором» в родной код платформы (Desktop, Web, Android, iOS, и т. д.) на другом языке программирования (C++, JS, PHP, Java, и прочие). Haxe легко выучить, если вы уже знакомы с объектно-ориентированными языками вроде тех же C++, Java, PHP, ActionScript3.
  • Транслятор кода, переводящий код с Haxe в код целевой платформы для последующей сборки или в байт-код виртуальных машин.
  • Стандартная библиотека, включающая работу с структурами и сериализацией данных, криптографией, файловой системой, базами данных и т. д. Также стандартная библиотека включает специфичное для платформ API, например, интерфейсы для работы с документом при трансляции в JavaScript.
