Введение в Haxe | 24.05.2018 |

Перевод статьи документации Haxe Language Features ( http://haxe.org/documentation/introduction/language-features.html) с официального сайта. Все права на оригинальный текст принадлежат авторам, все ошибки перевода на ответственности переводчика.

Абстрактные типы

Абстрактные типы это конструкции на этапе компиляции, которые представлены иначе во время выполнения. Это позволяет дать существующим типам совершенно новое значение.

Анонимные структуры

Данные могут быть легко сгруппированы в анонимные структуры, минимизируя потребность в небольших классах данных.

var point = { x: 0, y: 10 };
point.x += 10;

Включение в массив

Создавайте и заполняйте массивы быстро, используя циклы и логику.

var evenNumbers = [ for (i in 0...100) if (i%2==0) i ];

Классы, интерфейсы и наследование

Haxe позволяет структурировать код в классы, это делает его объектно- ориентированным языком. Поддерживаются общие связанные функции, известные из таких языков как Java, включая наследование и интерфейсы.

Условная компиляция

Условная компиляция позволяет компилировать специфичный код в зависимости от параметров компиляции. Это позволяет абстрагироваться от специфичных для платформ различий, но может использоваться и для других целей, таких как более подробная отладка.

#if js
 js.Lib.alert("Hello");
#elseif sys
 Sys.println("Hello");
#end

(Обобщенные) алгебраические типы данных

Структуры могут быть выражены через алгебраические типы данных (АТД), которые известны в Haxe как перечисления (enums). Кроме этого, Haxe поддерживает их обобщенный вариант (ОАДТ).

enum Result {
 Success(data:Array<Int>);
 UserError(msg:String);
 SystemError(msg:String, position:PosInfos);
}

Встраиваемые вызовы

Функции могут быть спроектированы как встраиваемые, позволяя вставлять их код по месту вызова. Это может значительно увеличить производительность без дублирования кода при помощи ручного встраивания.

Итераторы

Итерирование по набору значений, например, по элементам массива, очень легко реализуется в Haxe через итераторы. Пользовательские классы могут легко реализовать функциональность итераторов.

for (i in [1, 2, 3]) {
 trace(i);
}

Локальные функции и замыкания

Функции в Haxe не ограничены полями классов и могут быть легко объявлены в выражениях, позволяя создавать мощные замыкания.

var buffer = "";
function append(s:String) {
 buffer += s;
}
append("foo");
append("bar");
trace(buffer); // foobar
